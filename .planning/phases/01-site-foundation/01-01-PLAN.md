<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CDN -> Browser | Google Fonts loaded from external CDN |
| Markdown Content -> HTML | VitePress transforms Markdown to HTML (potential XSS if content is untrusted) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | Google Fonts CDN link | accept | Google Fonts is a trusted CDN with SLA; subresource integrity not supported for CSS imports. Risk is low (CDN compromise would affect millions of sites). |
| T-01-02 | Information Disclosure | .vitepress/config.ts | mitigate | Ensure config.ts contains no secrets (API keys, tokens). Config only has public site metadata. Verify no env vars with secrets are referenced. |
| T-01-03 | Spoofing | npm dependencies | mitigate | Use `npm ci` (not `npm install`) in CI to ensure reproducible builds from lockfile. Pin vitepress version in package.json with `^` semver to avoid unexpected major bumps. |
</threat_model>
