# Phase 1 — UI Design Contract

> Visual and interaction contract for the Site Foundation phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | VitePress default theme (extended via CSS custom properties) |
| Preset | not applicable (VitePress, not shadcn) |
| Component library | VitePress built-in components + custom Vue SFCs |
| Icon library | none (Phase 1 uses VitePress built-in icons; game-specific icons deferred to later phases) |
| Font | `'Noto Sans TC', 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif` |

**Implementation:** Override VitePress CSS custom properties in `.vitepress/theme/custom.css`. Import custom CSS in `.vitepress/theme/index.ts` alongside the default theme.

---

## Spacing Scale

VitePress controls layout spacing internally. The following scale applies to custom Vue components and content template layouts only.

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Compact element spacing, tag gaps |
| md | 16px | Default content element spacing, card internal padding |
| lg | 24px | Section padding within content templates |
| xl | 32px | Template section breaks (e.g., between Boss stats and strategy) |
| 2xl | 48px | Major content section dividers |
| 3xl | 64px | Hero section vertical padding on home page |

Exceptions: VitePress nav height remains at its default 64px. Sidebar width remains at default 272px. These are not overridden.

---

## Typography

VitePress provides its own typographic scale. The following overrides and additions define the contract for this project.

| Role | Size | Weight | Line Height | CSS Variable / Usage |
|------|------|--------|-------------|----------------------|
| Body | 16px | 400 (regular) | 1.7 | `--vp-font-family-base` override; default content paragraphs |
| Label / Caption | 14px | 400 (regular) | 1.5 | Badge text, table headers, frontmatter metadata, sidebar items |
| Heading (h2/h3) | 20px | 700 (bold) | 1.3 | Content section headings within templates |
| Display (h1 / page title) | 28px | 700 (bold) | 1.2 | Page titles, home page hero heading |

**Font family override:**

```css
:root {
  --vp-font-family-base: 'Noto Sans TC', 'Inter', -apple-system,
    BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;
  --vp-font-family-mono: 'JetBrains Mono', ui-monospace, 'Menlo', 'Monaco',
    'Consolas', monospace;
}
```

**Noto Sans TC loading:** Load via Google Fonts `<link>` in VitePress `head` config. Load weights 400 and 700.

### Accent Reserved For (explicit list)

1. **Brand color in nav** -- active nav item underline, site title hover
2. **Links** -- `--vp-c-brand-1` mapped to accent; text links in content
3. **Search highlight** -- matched search terms
4. **Primary action buttons** -- home page CTA buttons only
5. **Content template badges** -- Boss difficulty badge, equipment rarity badge (gold accent for legendary/epic)
6. **Sidebar active item** -- current page indicator

Accent is NOT used for: general borders, background tints on non-interactive elements, body text, headings, or decorative separators.

## Interaction Contracts

### Navigation
- **Top nav:** 8 items, horizontal scroll on smaller screens if needed. Active item has accent-colored underline (2px solid `--vp-c-brand-1`).
- **Sidebar:** Auto-generated from directory structure. Collapsible groups. Active item has accent left border (3px solid `--vp-c-brand-1`).
- **Mobile nav:** Hamburger menu at top-left. Sidebar slides in from left as overlay with backdrop.

### Search
- **Trigger:** Keyboard shortcut `Ctrl+K` / `Cmd+K`, or click search icon in nav.
- **Modal:** VitePress built-in search modal. Results highlight matched terms in accent color.
- **Empty state:** Shows "找不到相關內容" with suggestion copy.

### Theme Toggle
- **Position:** Top nav, right side, before social links.
- **Behavior:** Toggles `.dark` class on `html` element. Persists preference in localStorage.
- **Default:** Dark mode.

### Content Template Tables
- **Overflow:** Horizontal scroll on mobile for wide tables.
- **Header:** `--vp-c-bg-soft` background, 700 weight, 14px size.
- **Row hover:** Subtle background shift to `--vp-c-bg-soft` with 150ms transition.
- **Borders:** 1px `--vp-c-divider` between rows.

### Links (Cross-references)
- **Style:** Accent color, no underline by default. Underline on hover.
- **External links:** Append external link icon (VitePress default behavior).
- **Internal cross-links:** Same-tab navigation. Smooth scroll to anchor if same page.

## Accessibility Baseline

- **Language attribute:** `lang="zh-Hant"` on `<html>` element (set in VitePress config)
- **Color contrast:** All text/background combinations meet WCAG 2.1 AA (4.5:1 for body text, 3:1 for large text)
- **Focus indicators:** VitePress default focus rings preserved (not overridden)
- **Skip navigation:** VitePress provides built-in skip-to-content link
- **Semantic HTML:** Content templates use proper heading hierarchy (h1 for page title, h2 for sections, h3 for subsections)
- **Image alt text:** All template images require descriptive alt text in Traditional Chinese
- **Search keyboard access:** Full keyboard navigation in search modal
