# 精煉系統

了解裝備強化的完整流程，提升你的戰力上限。

## 基本說明

赤血沙漠的裝備強化主要透過**精煉（Refinement）**系統進行。精煉可提升武器和防具的基礎數值，最高可精煉至 +10 級。重要特點：精煉**不消耗金幣**，僅消耗材料。

## 精煉流程

1. 前往**鍛造師（Blacksmith）**— 第一位為埃爾南德城的圖爾納利（Turnali）
2. 選擇**「Refinement」**選項
3. 選擇要升級的裝備
4. 提交所需材料
5. 精煉完成，裝備等級提升

## 精煉等級與材料

精煉共 0 → 10 級（最高 +10）。

### 材料需求

| 裝備類型 | 所需材料 | 材料來源 |
|---------|---------|---------|
| 武器、板甲 | 礦石（鐵礦、銅礦及稀有變體） | 山壁/懸崖礦脈採集 |
| 木盾、弓 | 木材（Basic/Fine/Flawless Timber） | 砍伐樹木 |
| 皮革防具、飾品 | 皮革、小型/大型骨頭 | 獵殺動物掉落 |
| 布料、披風 | 羊毛、布料碎片、羽毛、象牙、蜘蛛絲 | 敵人掉落、探索 |

:::warning 注意
等級 5 以上的精煉需要**深淵神器（Abyss Artifacts）**作為額外材料，資源消耗量大幅增加。
:::

### 獨特武器精煉

- 非獨特裝備可使用**複製品**進行精煉
- 獨特武器需要**特定材料**（非通用材料），無法使用複製品替代

## 深淵核心系統（Abyss Core）

裝備插槽系統，可插入深淵核心提供特殊加成效果。

### 深淵核心效果

| 效果類型 | 說明 |
|---------|------|
| 攻擊速度加成 | 提升攻擊頻率 |
| 暴擊率提升 | 增加暴擊機率 |
| 元素傷害加成 | 附加元素屬性傷害 |
| 生命偷取 | 攻擊時回復生命值 |
| 範圍傷害附加 | 增加 AOE 效果 |

### 插槽系統

- 普通武器預設 **3 個插槽**
- 訪問**深淵女巫（Abyss Witch）**NPC 可花費 Abyss Cells 新增插槽（Create Socket 選項）
- 遊戲地圖散布 **5 位深淵女巫**，透過主線任務或特定位置解鎖（第三章後開始可用）
- **Abyss Cells 來源：** 擊敗敵人、完成任務、分解不要的深淵核心裝備

:::tip 早期建議
先完成 Rhett 與 Turnali 的支線任務，取得鎬頭和伐木斧，啟動資源採集，為精煉做準備。
:::

## 相關裝備

- [狼之劍（Sword of the Wolf）](/guide/equipment/sword-of-the-wolf) — 起始武器，可用標準礦石精煉

# 製作系統

了解裝備製作的材料需求與合成流程，打造專屬強力裝備。

## 基本說明

赤血沙漠擁有完整的製作系統，共有 **355 個製作手冊（Crafting Manuals）**涵蓋全遊戲可製作物品。

## 五大製作類型

| 製作類型 | 製作地點 | 製作內容 |
|---------|---------|---------|
| 裝備鍛造 | 鍛造師（Smithy）/ 裁縫（Tailor） | 武器、防具 |
| 煉金術 | 大釜（Cauldron） | 藥水、強化劑 |
| 烹飪 | 篝火/野外鍋 | 料理（增益效果） |
| 染料製作 | 染料站（Dye Station） | 裝備顏色染色 |
| 裝備強化 | 鍛造師 | [精煉升級](/guide/systems/enhancement) |

## 製作手冊取得

製作手冊散布於遊戲世界各處，取得後需交給對應的 NPC：

- **裝備手冊** → 交給裝備商、裁縫或鍛造師
- **煉金手冊** → 交給煉金師
- **烹飪手冊** → 交給廚師或自行使用

:::tip 收集建議
探索每個區域時注意搜刮寶箱和與 NPC 對話，許多製作手冊藏在不起眼的角落。
:::

## 資源採集

| 資源類型 | 採集方式 | 用途 |
|---------|---------|------|
| 礦石 | 採礦（Mining nodes — 山壁、懸崖） | 武器鍛造、板甲製作 |
| 木材 | 伐木（Logging） | 木盾、弓製作 |
| 食材 | 狩獵、採集 | 烹飪 |
| 皮革/骨頭 | 獵殺動物 | 皮革防具、飾品 |
| 特殊材料 | Boss 掉落、寶箱、任務 | 高階製作 |

:::info 資料待補充
各製作手冊的具體配方和材料需求尚在整理中，將於後續更新補充完整的製作配方表。
:::
