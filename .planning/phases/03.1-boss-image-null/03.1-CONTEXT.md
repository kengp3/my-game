<decisions>
## Implementation Decisions

### 圖片來源與授權
- **D-01:** 使用官方遊戲素材（官方網站、新聞稿、宣傳截圖）和遊戲內截圖為主要來源 — 攻略 Wiki 屬 fair use 範疇，標註來源即可
- **D-02:** 優先順序：Pearl Abyss 官方媒體素材 > 遊戲 Wiki 站（如 fandom） > 遊戲內截圖 > 社群分享圖片
- **D-03:** 每張圖片在 commit message 或圖片 alt text 中標註來源出處

### 圖片格式與規格
- **D-04:** 使用 WebP 格式 — 檔案較小、瀏覽器支援度已足夠（VitePress 靜態站不需 fallback）
- **D-05:** 角色與 Boss 圖片最大寬度 600px，裝備圖片最大寬度 400px — 平衡畫質與檔案大小
- **D-06:** 圖片壓縮至合理品質（WebP quality ~80）— 攻略 Wiki 不需無損畫質

### 圖片儲存路徑
- **D-07:** 存放於 `public/images/` 下按類型分目錄：`characters/`、`equipment/`、`bosses/`
- **D-08:** 檔名使用對應 Markdown 檔案的英文 slug（如 `kliff.webp`、`sword-of-the-wolf.webp`）— 與現有英文 URL slug 慣例一致

### 搜集範圍
- **D-09:** 僅處理現有 8 個 `image: null` 頁面 — 未來新增頁面時各自處理圖片
- **D-10:** 8 個目標頁面：kliff、damiane、oongka（角色）、example-equipment、sword-of-the-wolf、grey-wolf-leather-armor、pailunese-signet（裝備）、example-boss（Boss）

### Claude's Discretion
- 具體圖片搜尋關鍵字和搜尋策略
- 圖片裁切和調整方式
- 若找不到合適圖片時的替代方案（保持 null 或使用通用佔位圖）
- 是否需要額外的圖片優化工具

</decisions>
