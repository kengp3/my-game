<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Fextralife CDN → local /tmp | Untrusted external CDN serves PNG bytes over HTTPS |
| /tmp source → sharp → public/images | sharp re-encodes untrusted PNG into WebP (sanitizes metadata) |
| public/images → git commit | Binary files enter version control |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03.1-06 | Tampering | Fextralife CDN PNG downloads | mitigate | HTTPS enforced (all URLs start `https://`); curl `-f` flag rejects HTTP errors; PNG magic-byte verification in Task 1 rejects any response that is not a real PNG (e.g., HTML error page served with 200). |
| T-03.1-07 | Spoofing | Fextralife CDN MITM | accept | HTTPS + curl's default cert validation is sufficient for this low-value supply chain (wiki images, not executable code). No pinned cert. |
| T-03.1-08 | Information Disclosure | Downloaded PNGs may contain EXIF/metadata from Pearl Abyss | mitigate | sharp's `.webp({ quality: 80 })` pipeline strips all non-image metadata by default. No EXIF survives the conversion. |
| T-03.1-09 | Denial of Service | Malformed PNG exploiting libpng/libvips via sharp | mitigate | sharp 0.34.5 ships latest libvips (per npm metadata) with all known-CVE patches as of 2026-04-11. Source bytes come from a known public CDN, not user upload. |
| T-03.1-10 | Tampering | Stale Fextralife image (content changed since research) | accept | Attribution recorded in commit message (D-03); future content changes would require a new commit to update. Acceptable for a fair-use wiki. |
| T-03.1-11 | Repudiation | Source URL provenance | mitigate | All 8 source URLs embedded verbatim in plan 05's commit message (D-03 satisfies attribution requirement via Dimension 7 of Validation Architecture). |

Security enforcement scope: ASVS L1. No runtime attack surface — images are static assets served by GitHub Pages.
</threat_model>
