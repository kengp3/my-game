# Requirements: 赤血沙漠攻略 Wiki

**Defined:** 2026-04-10
**Core Value:** 提供玩家一站式完整攻略，讓任何進度的玩家都能找到所需資訊順利通關

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Site Infrastructure

- [ ] **SITE-01**: 使用者可瀏覽 VitePress 靜態網站，具有清晰的頂部導航和側邊欄
- [ ] **SITE-02**: 使用者可透過中文全文搜尋找到任何攻略內容
- [ ] **SITE-03**: 網站透過 GitHub Pages 自動建置部署，推送即上線
- [ ] **SITE-04**: 所有攻略頁面使用統一模板（Boss、裝備、區域各有專屬模板）
- [ ] **SITE-05**: 使用者可在手機/平板/桌面正常瀏覽所有內容（響應式設計）

### Main Story

- [ ] **STORY-01**: 使用者可按章節順序查閱主線劇情流程與推進指引
- [ ] **STORY-02**: 每章攻略包含關鍵選擇點說明與建議推進順序
- [ ] **STORY-03**: 使用者可查閱所有支線任務的觸發條件、流程和獎勵
- [ ] **STORY-04**: 使用者可查閱隱藏任務和彩蛋的觸發條件與完成方式

### Combat & Bosses

- [ ] **COMBAT-01**: 使用者可閱讀戰鬥系統基礎教學（操作、連招、防禦機制）
- [ ] **COMBAT-02**: 使用者可查閱每個 Boss 的詳細攻略（弱點、攻擊模式、推薦裝備）
- [ ] **COMBAT-03**: 使用者可查閱高難度挑戰的專門攻略與技巧
- [ ] **COMBAT-04**: 每個 Boss 攻略頁連結到相關掉落裝備和所在區域

### Characters & Equipment

- [ ] **CHAR-01**: 使用者可查閱各角色/職業的介紹、技能樹和建構推薦
- [ ] **CHAR-02**: 使用者可瀏覽完整裝備圖鑑（武器、防具、飾品），含屬性和取得方式
- [ ] **CHAR-03**: 使用者可查閱裝備強化和製作系統的流程說明
- [ ] **CHAR-04**: 裝備頁面連結到取得地點（Boss 掉落、商店、製作）

### Maps & Exploration

- [ ] **MAP-01**: 使用者可查閱每個區域的地圖指南（敵人、NPC、重要地點）
- [ ] **MAP-02**: 使用者可查閱隱藏區域位置和進入方式
- [ ] **MAP-03**: 使用者可查閱寶箱位置和內容物
- [ ] **MAP-04**: 使用者可使用收集品指南追蹤全收集進度

## v2 Requirements

### Interactive Features

- **INTER-01**: 使用者可使用互動式地圖標記已探索區域
- **INTER-02**: 使用者可使用 Build 計算器模擬裝備/技能配置
- **INTER-03**: 使用者可標記已完成任務追蹤進度
- **INTER-04**: Boss 攻略嵌入示範影片

### Community

- **COMM-01**: 使用者可在攻略頁面留言補充資訊
- **COMM-02**: 多語系支援（簡體中文、英文）

## Out of Scope

| Feature | Reason |
|---------|--------|
| 使用者帳號系統 | v1 為靜態網站，不需要帳號功能 |
| 即時聊天/論壇 | 可導向既有社群（巴哈姆特、Reddit） |
| 遊戲資料自動抓取 | 技術複雜度高、法律風險 |
| 影片上傳/託管 | 可嵌入外部影片連結替代 |
| 遊戲內掛 (mod) 教學 | 非攻略核心範圍 |
