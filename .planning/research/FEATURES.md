# Features Research — 赤血沙漠 Game Guide Wiki

## Table Stakes (Must Have)

| Feature | Complexity | Description |
|---------|-----------|-------------|
| 全文搜尋 | Low | 玩家能快速找到需要的攻略內容 |
| 響應式設計 | Low | 手機/平板/桌面都能正常瀏覽 |
| 清晰導航 | Low | 側邊欄 + 麵包屑 + 分類目錄 |
| 主線攻略 | Medium | 完整主線劇情流程與任務指引 |
| Boss 攻略 | Medium | 每個 Boss 的弱點、戰術、推薦裝備 |
| 角色/職業指南 | Medium | 各角色/職業的技能樹、建構推薦 |
| 裝備資料庫 | High | 所有裝備的屬性、取得方式、比較 |
| 地圖指南 | Medium | 各區域的敵人、寶箱、NPC 位置說明 |

## Differentiators (Competitive Advantage)

| Feature | Complexity | Description | Dependencies |
|---------|-----------|-------------|--------------|
| 互動地圖 | High | 可標記、篩選的互動式地圖 | 地圖基礎資料 |
| Build 計算器 | High | 模擬裝備/技能配置的工具 | 裝備資料庫 |
| 進度追蹤器 | Medium | 玩家可標記已完成的任務/收集品 | 主線攻略 |
| 影片嵌入 | Low | Boss 戰鬥示範影片 | Boss 攻略 |
| 社群留言 | Medium | 玩家可分享補充攻略 | — |

## Anti-features (v1 Not Building)

| Feature | Reason |
|---------|--------|
| 使用者帳號系統 | 過度複雜、v1 專注於內容本身 |
| 即時聊天 | 非攻略核心功能、維護成本高 |
| 多語系支援 | v1 專注繁體中文、後續可擴展 |
| 遊戲資料自動抓取 | 需要逆向工程、法律風險 |
| 論壇/討論區 | 可導向現有社群（如 Reddit、巴哈姆特） |

## Content Structure

### 核心內容分類

1. **新手入門** — 遊戲基礎、操作說明、初期建議
2. **主線劇情** — 章節制劇情流程攻略
3. **支線任務** — 所有支線任務觸發條件與獎勵
4. **Boss 圖鑑** — 每個 Boss 的詳細攻略
5. **角色/職業** — 職業介紹、技能樹、建構指南
6. **裝備圖鑑** — 武器、防具、飾品資料
7. **地圖探索** — 各區域地圖、隱藏物品、NPC
8. **系統機制** — 戰鬥系統、強化系統、製作系統等

### Feature Dependencies

```
新手入門 (獨立)
  ↓
主線劇情 → 支線任務
  ↓
Boss 圖鑑 ← 角色/職業
  ↓
裝備圖鑑 ← 地圖探索
  ↓
系統機制 (貫穿所有)
```
