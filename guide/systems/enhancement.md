# 精煉系統

了解裝備強化的完整流程，提升你的戰力上限。

## 基本說明

赤血沙漠的裝備強化主要透過**精煉（Refinement）**系統進行。精煉可提升武器和防具的基礎數值，最高可精煉至 +10 級。重要特點：精煉**不消耗金幣**，僅消耗材料。

## 精煉流程

1. 前往**鍛造師（Blacksmith）**— 第一位為埃爾南德城的圖爾納利（Turnali）
2. 選擇**「Refinement」**選項
3. 選擇要升級的裝備
4. 提交所需材料
5. 精煉完成，裝備等級提升

## 精煉等級與材料

精煉共 0 → 10 級（最高 +10）。

### 材料需求

| 裝備類型 | 所需材料 | 材料來源 |
|---------|---------|---------|
| 武器、板甲 | 礦石（鐵礦、銅礦及稀有變體） | 山壁/懸崖礦脈採集 |
| 木盾、弓 | 木材（Basic/Fine/Flawless Timber） | 砍伐樹木 |
| 皮革防具、飾品 | 皮革、小型/大型骨頭 | 獵殺動物掉落 |
| 布料、披風 | 羊毛、布料碎片、羽毛、象牙、蜘蛛絲 | 敵人掉落、探索 |

:::warning 注意
等級 5 以上的精煉需要**深淵神器（Abyss Artifacts）**作為額外材料，資源消耗量大幅增加。
:::

### 獨特武器精煉

- 非獨特裝備可使用**複製品**進行精煉
- 獨特武器需要**特定材料**（非通用材料），無法使用複製品替代

## 深淵核心系統（Abyss Core）

裝備插槽系統，可插入深淵核心提供特殊加成效果。

### 深淵核心效果

| 效果類型 | 說明 |
|---------|------|
| 攻擊速度加成 | 提升攻擊頻率 |
| 暴擊率提升 | 增加暴擊機率 |
| 元素傷害加成 | 附加元素屬性傷害 |
| 生命偷取 | 攻擊時回復生命值 |
| 範圍傷害附加 | 增加 AOE 效果 |

### 插槽系統

- 普通武器預設 **3 個插槽**
- 訪問**深淵女巫（Abyss Witch）**NPC 可花費 Abyss Cells 新增插槽（Create Socket 選項）
- 遊戲地圖散布 **5 位深淵女巫**，透過主線任務或特定位置解鎖（第三章後開始可用）
- **Abyss Cells 來源：** 擊敗敵人、完成任務、分解不要的深淵核心裝備

:::tip 早期建議
先完成 Rhett 與 Turnali 的支線任務，取得鎬頭和伐木斧，啟動資源採集，為精煉做準備。
:::
