# 製作系統

了解裝備製作的材料需求與合成流程，打造專屬強力裝備。

## 基本說明

赤血沙漠擁有完整的製作系統，共有 **355 個製作手冊（Crafting Manuals）**涵蓋全遊戲可製作物品。

## 五大製作類型

| 製作類型 | 製作地點 | 製作內容 |
|---------|---------|---------|
| 裝備鍛造 | 鍛造師（Smithy）/ 裁縫（Tailor） | 武器、防具 |
| 煉金術 | 大釜（Cauldron） | 藥水、強化劑 |
| 烹飪 | 篝火/野外鍋 | 料理（增益效果） |
| 染料製作 | 染料站（Dye Station） | 裝備顏色染色 |
| 裝備強化 | 鍛造師 | [精煉升級](/guide/systems/enhancement) |

## 製作手冊取得

製作手冊散布於遊戲世界各處，取得後需交給對應的 NPC：

- **裝備手冊** → 交給裝備商、裁縫或鍛造師
- **煉金手冊** → 交給煉金師
- **烹飪手冊** → 交給廚師或自行使用

:::tip 收集建議
探索每個區域時注意搜刮寶箱和與 NPC 對話，許多製作手冊藏在不起眼的角落。
:::

## 資源採集

| 資源類型 | 採集方式 | 用途 |
|---------|---------|------|
| 礦石 | 採礦（Mining nodes — 山壁、懸崖） | 武器鍛造、板甲製作 |
| 木材 | 伐木（Logging） | 木盾、弓製作 |
| 食材 | 狩獵、採集 | 烹飪 |
| 皮革/骨頭 | 獵殺動物 | 皮革防具、飾品 |
| 特殊材料 | Boss 掉落、寶箱、任務 | 高階製作 |

:::info 資料待補充
各製作手冊的具體配方和材料需求尚在整理中，將於後續更新補充完整的製作配方表。
:::
